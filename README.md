# Building from source

## Dependencies

* [Go 1.5.1](http://golang.org/) or 1.4
* [Git](http://git-scm.com/) 
* [Bats](https://github.com/sstephenson/bats)
* [fdm](https://github.com/pki-io/fdm) (just a bash script)

## Build

    $ make && sudo make install
